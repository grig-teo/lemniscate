Lemniscate

// a self-improving codebase

Connect GitHub, GitVerse or GitLab, plug in your own LLM, and Lemniscate analyzes your repositories, then proposes and implements improvements, features and fixes as pull requests — with optional LLM review and auto-merge.

$

This script installs Docker if it is not installed yet, then deploys Lemniscate via Docker Compose.

your own LLM

Any OpenAI-compatible endpoint — OpenAI, vLLM, Ollama, LM Studio — configured in the UI.

PRs, not patches

Proposals and fixes land as branches and pull requests on your git host.

review & auto-merge

Optional second-pass LLM review, with automatic merge when it approves.

live console

Watch the agent think, edit, commit and push in real time over SSE.
